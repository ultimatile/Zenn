# GLOSSARY.md（用語・表記の方針）

全用語を列挙する辞書ではない。**書き方が割れそうな語にだけ**集中して方針を固定する。
新しく迷う語が出たら、まず一般方針に当て、人名由来語は弁別軸で判断する。

**運用原則:** このガイドラインで判断できる語はそれに従う。判断できず **新造（新しい訳語・用語の造語）が必要な語** は、勝手に確定せず **具体案を添えてユーザーに確認**してから採用する。

## 英単語の基本方針

優先順位:

1. **日本語に訳す**（定着した訳語があればそれを使う）。
2. 適切な訳語・定着語がなければ**カタカナ**。
3. 訳すと意味がぶれる/損なう技術用語は**原語（英語）のまま**。例: `load-bearing`、（`thiserror` 文脈の）`reporter`。

ただし **人名は常にラテン文字フルスペル**。カタカナにしない（Gauss / Hermite / Jacobi / Gram / Hamilton）。
この綴りルールはカタカナ化より優先し、**定着語の中の人名部分にも適用する**（後述）。

## 人名由来のテクニカルターム（最重要・最も割れる）

**綴り:** 人名部分は常にラテン文字。ガウス→Gauss、エルミート→Hermite、ヤコビ→Jacobi。

**語の作り方（弁別軸は「定着しているか／新造になるか」）:**

- 定着した概念は **定着語を使い、人名部分だけラテン綴り**にする。
  - エルミート行列 → **Hermite行列**
  - ガウス分布 → **Gauss分布**（訳語の「正規分布」を使うのも可）
  - ヤコビ行列 → **Jacobi行列**
- 定着していない概念に「人名＋名詞」を **新造しない**。一般方針/複合語で処理する。
  - 例: "Gaussian matrix" を「Gauss行列」とは書かない（その語は定着していない）。
  - `Gaussian〜` を禁じたのは**未定着の場合**の話。Gauss分布のような定着語は可。
- 対象そのものを指す独立した `-ian` 名詞で、頭名詞を持たないもの → **原語（-ian）のまま**。
  - **Hamiltonian**（演算子/関数そのもの。「Hamilton行列」等にしない）。
- 行列と行列式が同じ `-ian` 語に同居して衝突するもの → 行列には「人名＋行列」を使い、裸の `-ian` を行列の呼称に使わない（`-ian` は本来その**行列式**）。原文で出たら方言として注釈。
  - **Jacobian**: 行列の意味では**禁止** → **Jacobi行列**。原文で行列の意味の "Jacobian" が出たら「本来は Jacobi 行列式の意」と方言注釈する。

## Gramian（文脈依存）

分野間で意味が衝突するため、文脈で切り替える。

- **線形代数**: 行列は **Gram行列**、行列式は **Gram行列式**。裸の "Gramian" は本来「行列式」を指す（Jacobian と同じ構図）ので、行列の呼称として出てきたら方言として注釈する。
- **制御理論**: **可制御/可観測Gramian** は行列そのものを指す定着名（A\*A の行列式ではない積分行列）なので矯正せず、ラテン綴りで許容する。
- 上記2文脈に当てはまらない新造ケースは、運用原則どおり提案してユーザーに確認する。
